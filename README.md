# Pwr
This project prints power statistics for GNU/Linux and FreeBSD

## Install
### GNU/Linux
Clone & cd into repo, then use GNU Make to install:
```
make && sudo make install
```

### FreeBSD
Install dependencies:
```
pkg install libexplain
```

Clone & cd into repo, then use GNU Make to install:
```
gmake && sudo gmake install
```
